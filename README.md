# C Learning

This repository contains basic exercises and practice programs written in C.

## Content

- Conditional statements (if/else)
- Loops (while)
- Basic mathematical operations
- Input and output (scanf/printf)

## Exercises

- Discount calculation
- Finding the largest and smallest values
- Counting even numbers and calculating the average of odd numbers
- Square root calculation (custom implementation)

## Goal

Build a solid foundation in programming logic and core concepts of the C language.
